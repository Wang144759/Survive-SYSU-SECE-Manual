# 公选课大全
这里将会贴出一份往届学长学姐（并非本指南）总结出的公选课推荐作为参考，那些成名已久的公选课往往都有其特色。

[中山大学公选课推荐](https://pan.quark.cn/s/d500823fdd1f)

:::tip
此推荐已经有一段时间没有更新了。公选课的设置每年的情况都会有所变动，特别是那些新开的课程从授课进度到结题方式以及给分都可能会和往届参考消息有所不同。
:::

这里给出选课策略方面的一些建议。往往大一大二的时候同学们会着急修满公选学分而紧张抢课，特别是公共艺术课非常难选。实则如果你没有特别的打算，提前修满公选学分这个选择并不一定有你们想象中那么好。原因很简单：大四特别是大四下的时候公选课简直是点击就送，并且有时候一些比赛或者奖项会直接给你公选学分。

对于有可能会出国留学或者正在犹豫留学相关的同学，这类同学往往需要尽可能选择得分较高的课程，如果没有抢到这类课程不必急着修满公选学分，到需要刷分的时候再选择。

对于计划本科就业的同学，这类同学确实需要尽早修完公选学分，不过不必在意课程的给分，凭自身规划选择即可。

对于打算保研的同学，本院保研在绩点方面只看必修课和专业选修课（大二才有），所以公选课凭自身规划选择即可，不过早点修完未来可以提前进组。

对于计划考研的同学，这类同学其实并不一定需要早点修完学分，考研途中不时上课放松一下也是一种选择，并且可以大四下再修公选。
