# 程序设计
电通的程序设计其实分为`C`语言和`C++`两部分，采用的教材都是谭浩强的红皮书。这两门语言在语法上非常类似，从课程学习来说最大的区别就是`C++`提供了更为标准的面向对象机制以及提供了非常多的`STL`，实际到应用中其实嵌入式更多会用到C，而游戏、音视频开发等更多用到`C#/C++`，`C++`的`STL`库对于刷题而言是必须掌握的一环。`C++`对于许多深造方向也十分重要，如比较火的`AI infra`等需要高性能的场景。但是总得来说由于过于底层以及`C++`体系过于庞杂，企业中用到`C++/C`作为产品主力的并没有想象中那么多，不是特别推荐作为找工主攻方向。
