# 不为人知的小技巧
和之前同学们人生经历不一样的是。中大录取通知书其实是张**兑奖券**，没人能具体评估它价值几何。大学就是这张兑奖券的兑换处，但是它的兑奖窗口往往藏在不为人知的地方，同时往往需要你付出**一定的努力**才能用它得到你想要的东西。

众所周知，你电通的课程多而杂，因此我们不得不考虑这样一个问题：
```
 学校的课要不要上，所有领域的知识真的都需要掌握吗？
```
也许不同人会有不同的答案，有一些学习比较专注或者死板的同学可以保证自己每节课都能认真听讲，课后认真完成作业并在期末拿到一个比较好的分数。但是对于另一些同学而言，如何**同时处理**不同的事情就需要你小心衡量了。

就我观察而言，对于一些**天赋异禀**的同学，他们平时除了拿平时分几乎完全不学，只需要在期末前两三个周从头开始把课件学一遍作业重新做一遍，即可轻松达到保研标准线之上（电通综测大概`前30%`的同学有较大保研机会），然后把多余的时间去完成一些他们看来更有意义的事情。

### 一、不推荐使用但可能会救你一命的小技巧

:::tip
（特别是你要翘课去实习或者打比赛的时候）
:::

* 关于点名的规律：第一节课肯定是要点名的，后面就不一定了。站在老师的角度上看点名这件事情，当老师发现来的人非常少的时候，有**很大概率会选择点名**，防止到的同学越来越少。往往前两节课是有很大概率点名（当然第一节课肯定是要点名的），后面的课就不一定了。

* 一般**学期中后**的时候是**最容易**突然点名的，因为同学流失得越来越多，中后期的时候，老师会发现同学们的积极性越来越低，这时候就要慎重考虑是否点名了。有时候实在有事第一节课是可以不去选择第二节课课间和老师battle的。

* 学院的活动往往都有**位置签到**，有时还有现场**纸质题目**。但是其实不去也不会有什么实质性影响，最多被辅导员问候一下。不过和辅导员交流多了其实也还好。



### 二、关于奖学金

> 学院设有多种奖学金，每年有大概 **50%** 的同学能获得奖学金，因此大学期间获得奖学金的概率是比较大的。

主要奖学金类型如下：

*   **学院专项奖学金**：主要包含学术创新、道德风尚、学科竞赛、文体艺术、学业进步五类，这类奖学金的占比较大。
*   **学校优秀学生奖学金**：包括一等、二等、三等奖学金（涵盖国家奖学金、国家励志奖学金等），这类奖学金每年级有**一百人**左右能拿到。
*   **社会捐赠奖学金**：即那些奖学金名称上带有个人或企业名字的，这种名额通常很少，获取概率较小。

:::tip
**关于助学金与补贴：**

1.  **助学金**：主要为家庭经济困难的学生准备。一般来说，只要家庭经济状况中等以下，且非公职人员家庭，基本都能申请到。
2.  **就业补贴**：学校可能会有一些帮助应届生就业的补贴。虽然材料齐全、符合条件就有概率拿到，但流程通常比较繁琐，具体需要看当年的具体政策。
:::

### 三、如果真的不幸挂科了怎么办
* 如果你真的挂科了，我只能表示非常遗憾。中大毕业的难度极低，具体到电通只需要你毕业有`2.0/5.0`的均绩即可。

* 然而中大挂科的后果是 **相对严重** 的，因为在中大的成绩单上挂科补考和重修的成绩 **并不会直接覆盖原来的成绩**，而是会表明这是你挂科或者重修得来的成绩。这就意味着你挂科的成绩会 **直接影响** 到你以后申请的学校，甚至影响到你以后的工作。

* 所以，如果你真的挂科了，**保研、评奖评优、央国企、交换推荐** 都会受到不同程度的影响。这其中只有**私企就业和考研**受到的影响最小，考研本院初试复试都几乎完全不看绩点，选择就业更是无人在意你的绩点和你学了哪些课程。

* 最可怕的是，电通的课程安排可以说是一环套一环，如果你其中有一两门没搞懂，或者突然缺了几节课，那么你的学习进度大概率就会卡在那里，直到你将它从那个阻塞的情况中解脱出来。
:::tip

:::
