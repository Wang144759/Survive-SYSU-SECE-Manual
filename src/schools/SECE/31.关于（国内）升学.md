# 关于国内升学  
保研与考研路径说明

国内升学本质上是一种**长期投入换取未来选择空间**的决策。相比直接就业，继续深造在短期内的回报并不占优，但若具备条件，合理规划国内升学路径，往往能显著拓展后续发展的上限。

需要明确的是，选择保研还是考研，通常并非临近毕业才作出的决定。多数情况下，**是否具备保研竞争力在大一结束后已有较为清晰的判断**。因此，尽管两条路径在时间线上有所交叉，其适用人群与准备重点存在本质差异，本文仍将二者放在同一框架下说明。

从具体形式上看，国内升学主要涉及以下几种情况：
- 保研本校 / 保研外校（以夏令营与预推免为核心）
- 考研本院 / 考研外院或外校

:::tip
电通院的实际执行规则为：**每位导师在同一届中最多只能接收 1 名保研学生**。  
这意味着本校保研在导师层面存在明显的名额约束，需尽早确认意向与可行性。
:::

## 1.1 国内升学（保研 / 考研）

国内升学主要包括 **保研** 与 **考研** 两条路径，其共同特点是时间节点高度集中、关键窗口期短，一旦错过往往难以补救。

### 保研时间线（以外校保研为典型）

- **大三上学期**  
  明确保研资格与学院排名，开始筛选本校或外校的潜在导师。
- **大三下学期（3–6 月）**  
  准备个人简历、成绩单、科研或竞赛材料，密切关注各高校夏令营通知。
- **大三暑假（7–8 月）**  
  **参加夏令营（核心阶段）**，争取“优秀营员”或明确的导师意向。
- **大四上学期（9–10 月）**  
  参加预推免面试，完成教育部推免系统填报与最终确认。

### 考研时间线

- **大三下学期**  
  确定报考院校与专业，启动数学、英语等科目的系统复习。
- **大四上学期（9–10 月）**  
  完成考研报名，进入政治与专业课的强化与冲刺阶段。
- **大四上学期（12 月）**  
  全国硕士研究生招生考试初试。
- **大四下学期（次年 3–4 月）**  
  复试（笔试 + 面试）及最终录取结果确认。

:::tip
国内升学的整体特征可以概括为：**节奏快、窗口期短、信息不对称严重**。  
在保研路径中，导师的研究方向与个人偏好往往具有决定性影响，因此提前获取可靠信息、建立有效沟通渠道，是能否成功推免的关键因素之一。
:::
