
# 关于转专业

中大近几年的转专业政策比之前松不少，自从高松校长上台后，中大无论是转入还是转出的门槛，在全国的 985 高校里都属于相对友好的。

> 如果说高考报名志愿的过程很大程度上决定了一个人的职业生涯，那么转专业就很大程度上是同学们换赛道代价最小的机会。

## 1. 基本政策与流程

根据《中山大学本科生申报转专业常见问题解答》，以下是必须知道的官方规定：

### 1.1 时间节点
*   **频率**：每学年统一办理一次（大一、大二都有机会）。
*   **时间**：通常在 **春季学期（4月-5月）**。
    *   4月上旬：公布计划。
    *   4月下旬：教务系统报名。
    *   5月：各院系组织考核（笔试/面试）。
    *   6月：公布名单与学籍变动。

### 1.2 资格限制
虽然政策宽松，但以下情况**不予转专业**（“五不转”）：
1.  招生时明确规定不能转专业的（如部分强基计划、定向生等）。
2.  新生入学未满一学期的。
3.  **已经转过一次专业的**（每个人只有一次机会）。
4.  处于退学边缘（试读期）或休学期间的。
5.  其他特殊规定。

:::tip
**关键点**：中大转出**无限制**（不需要原院系盖章同意，只要想走都能走），难点全在于**转入**（目标院系是否收你）。
:::

## 2. 考核与录取逻辑

转专业并非只有“成绩好”这一条路，不同院系的考核方式差异巨大：

*   **笔试+面试**：热门院系的标配（如岭南、法学、计算机、医学）。
*   **仅面试**：大多数工科或冷门文科专业。
*   **是否降级**：由接收院系决定。跨度大的（如文转理，或大二转入）通常需要降级（转入大一），这意味着你要多读一年本科。

### 2.3 如何判断难度（数据分析策略）

不要只看“计划招收人数”，要看**报录比**。建议通过教务系统搜索往年的“通过名单”与“计划表”进行对比：

```text
难度判断公式：
1. 名单人数 < 计划人数：竞争极小，只要过了及格线甚至只要报名就能进。
2. 名单人数 = 计划人数：存在筛选，需要进一步评估报录比。
```

## 4. 避坑指南与建议

### 4.1 跨校区问题
中大有三校区五校园（广州南/北/东，珠海，深圳）。
*   **警惕**：转专业后可能涉及跨校区搬迁。例如从南校转到深圳校区，生活环境、宿舍条件会有巨大变化，且不同校区的培养方案可能不同。如果转入跨度大，需要补修大一课程，可能会导致大二大三课表爆炸，甚至需要延毕。

### 4.2 信息收集
1.  **官方渠道**：教务部官网、各学院官网（一定要看去年的考核方案）。
2.  **民间渠道**：
    *   校园 App “校园集市”：能第一时间看到各路小道消息。
    *   咨询师兄师姐：特别是已经转成功的人，他们的面试经验（问什么问题、考什么数学）价值连城。

:::tip
除非是为了热爱，不然转专业一般都是为了`更好的发展`。在这样的前提下考虑转专业，你背后需要巨大的付出。转专业总得来说**对绩点的要求是非常高的**，如果想要转专业请提前刷好`GPA`，认真准备，通过考试并不难。
:::
