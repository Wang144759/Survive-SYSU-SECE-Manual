# 关于生活

这一部分我主要想谈谈大学的生活，主要从`务实`和`务虚`两个方面来谈。

## 一、务实

### 1. 关于宿舍

* 对于一些同学而言，大学之前是没有住校的经历的。除了在课堂上进行学习以外，宿舍是大家待的时间第二长的地方，你和你室友的相处将会**持续三到四年**。

>因此提前对室友进行甄别是远比很多同学刚入学或者搬校区的时候预想的所更重要的事情，这件事情做得好的话你将会收获一些能够持续相当长的一段时间的友情。

* 如果实在运气不好的话，室友可能甚至无法和你做成最基本的朋友，如果你们的生活习惯有很大差距，那更是雪上加霜。

* 这个时候**合理的宿舍规则**将会是你们平稳度过剩下时学校时光的重要保障。常见的宿舍规则包括：

```宿舍规则
1. **作息与安静**

* 约定23:00熄灯，宿舍需保持安静。
* 夜间（尤其是熄灯后）不得大声喧哗或打扰他人。

 2. **安全与门禁**

* 不得擅自改动电路、门锁或宿舍设施。
* 不在宿舍使用电烙铁、热风机等危险电子工具。

3. **卫生与整洁**

* 宿舍成员应轮流打扫或各自负责各自卫生，保持 地面、桌面整洁。
* 不得随意堆放垃圾或食物残渣，防止虫害。
* 公共区域（如走廊、卫生间）定时清洁。

4. **用电与电器**

* 禁止私拉电线或使用大功率电器（如电暖器、电磁炉等），防止火灾。
* 离开宿舍应关闭电源，节约用电。

5. **文明与行为**

* 尊重室友，避免争吵或侵犯隐私。
* 不在宿舍内赌博、酗酒或进行违法行为。
* 讲究公共道德，遵守宿舍规定与学校管理制度。
```

### 2. 关于交通

在南校校园内，步行和自行车是我较为推荐的通行方式。而对于深圳校区的同学而言，几乎所有同学都会选择购置一辆电动车，还有一些热爱运动的同学会选择购买一辆质量较好的公路车或山地车。

:::tip
虽然网购电动车下单时似乎较为方便，价格较为便宜，但实际上我更推荐直接在学校附近寻找一家电动车专卖店挑选，在广东炎热的天气下如果没有很好的售后，你将会感到非常痛苦。
:::

### 3. 关于恋爱

国人的教育里面很多孩子是缺爱的，我衷心的希望天下有情人终成眷属。但非常遗憾的是，由于网络上愈发激烈的男女对立等一系列原因，男女关系已经变成了我看不懂的样子。

对于一些同学而言， 如果你光是和异性交流都经常感到很奇怪和别扭，那就说明有许多观念上的差异需要被改变，不然你将大概率最终不得不采取长时间远离异性的下策。
:::tip
作为成年人必须知道，任何社交情况下，极端的言论和情绪都会摧毁每一个人。
:::

### 4. 关于学校的食堂

整体来说中大的食堂水平参差不齐，总体来说并不算很好，但放在广深珠这种较发达城市中其实也并不能算贵。学校的食材并不会出现份量克扣的情况，南校的窗口种类较为丰富，在许多论坛上甚至可以搜到中大学子对于各菜式的打分和评价。

不过具体到深圳校区的食堂，特别是对外地的同学而言，**它无法给人享受食物的感觉**，特别是早餐。作为替代，可以选择去便利店或者校外购买所需。

### 5. 关于锻炼与健康管理

- **健康的身体**是人生最大的资本。在众多锻炼方式中，跑步无疑是成本最低且效率极高的选择。不过，中强度运动器材种类丰富，建议大家多尝试不同类型的运动，找到最适合自己的方式。可以称得上福利的一点是深圳校区，每天早上泳池人都比较少，并且校内游泳免费。

- **充足的睡眠、早睡早起、规律的三餐**，比任何补品都更有效。

然而现实是许多人连早睡早起都难以坚持。宿舍晚上不断电的政策虽有其便利，但也让不少人深受其扰。建议同宿舍成员约定一个固定的熄灯时间（如11点或12点），并保持安静，共同营造良好的休息环境。

### 6. 关于租房

租房可以去小红书上找适合租房的地方，然后找房东/中介沟通。和别人合作、交流、相处也是一种能力，关键是对于多数人而言它的培养其实并没有你想象的那么难。

:::tip
作为参考，在深圳的城中村中租一个`40-50平米`的单间一个月在`1500 RMB`左右，公寓类的单间则是在`2000 RMB`左右，明显高于这个价格则显得不太值得。一般租房是半年起租，若是小于半年大多属于短租的情况，房东一般会要求支付`200-500元`的短租费。
:::
### 7. 快递和外卖

*   **取件方式**：快递和外卖取件方式菜鸟驿站需要扫身份码出库。
*   **异常情况**：如果在非淘系 APP 购买，有可能遇到 `收货电话号码为虚拟号` + `无法获取取件码` 的情况，这时唯一忠实的溯源依据就是完整的快递单号，在菜鸟裹裹上可以查到取件码。
*   **安全提示**：及时取外卖！虽然我自己没有经历过，但是常常在校内论坛发现有人被偷外卖。

## 二、务虚

人生是漫长的又是短暂的，不管你选择了什么道路，现在看来艰难的抉择都只是人生的一小部分。

有的理工科学生除了上学就是打游戏看动漫小说，人际交往几乎为零。问他对于什么感兴趣，答曰不知道。问他对自己的未来有什么规划，对于社会经济政治有什么看法，回答则是没有什么看法。这样的人格很难说是健全的，同时这样的人生来也是悲惨的。

大学生可以分为四种，一无所有而迷茫的人，不知道想要什么但努力的人，知道自己想要什么的人，以及什么都有的人。从大学入学开始，同学们的生活和认知就有着极大的区别。
### 九个自问
如果你觉得对自己想要什么感到迷茫，对自己是一个什么样的人并不清楚，不妨问问自己如下的问题：
```
1.请你介绍一下你自己，你是个什么样的人？

2.你有什么理想吗？这个理想是怎么形成的？

3.你理想的伴侣关系是什么样的？你自己在这个伴侣关系中扮演什么样的角色？要承担什么样的责任？

4.你理想的事业是什么，你正在做的工作符合你的事业理想吗？这份工作对你的意义是什么？

5.你对亲子关系怎么看？对你来说，什么是一个理想的父亲（母亲），你期望自己成为这样一个理想父亲（母亲）吗？

6.你对钱怎么看，你认为赚到多少钱是足够的？如果你明天一早醒来，已经有足够的钱，你将会如何继续安排自己的生活？

7.你的择友标准是什么？什么样的人你会愿意交往，什么样的人你会拒绝和他交往？

8.你对死亡怎么看？你希望自己活到多少岁，你准备怎么度过从现在到死亡的这段时间？如果你要立遗嘱，这份遗嘱会怎么写？

9.你对于学习的态度是怎样的？你学习的目的是为了什么？

对于我而言，学习一方面是为了让自己学到更多专业知识，让我未来能够更好地生存；
另一方面，学习是为了让我变成一个更酷的人，这两个，可以说是我学习的终极目标了吧。

> ————以上的问题部分摘自
>《很少人能顺畅回答这9个问题——心理治疗刚开始医生常常会先问你的 》
> 我这里并不是想要推荐大家去看这本书
>实际上对于世界上多数人而言，将自己有限的精力耗费到以后并不会用到的东西上是一个不够理智的选择。
```
> 这些问题虽然看起来有些过于宏大沉重，可是对于一个成年人而言，这些问题本身便是来到这个世界上的人迟早要面对的。

- 每个人都会经历那么一段迷茫的时期，但当你真正能回答这十个问题，也许就不会那么迷茫了。
